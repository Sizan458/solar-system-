# 🌌 Solar System Simulation

A visually stunning, interactive 2D/3D visualization of our solar system. This project brings the cosmos to your browser, featuring realistic orbital motions, planet details, and a smooth user experience.

## 🚀 Features

- **Realistic Orbits:** Planets revolve around the Sun with synchronized speeds and paths.
- **Interactive UI:** Click on planets to view specific data and zoom in on their details.
- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Visual Effects:** Includes starry backgrounds, planet shadows, and smooth CSS/JS animations.
- **Educational:** Provides quick facts about each celestial body (Mass, Diameter, Distance from Sun).

## 🛠️ Built With

* [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) - Structure of the simulation.
* [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) - Styling and keyframe animations.
* [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - Orbital logic and interactivity.
